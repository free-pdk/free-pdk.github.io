PMS15A

  • Instruction Set: PDK13
  • ROM Size: 0.5 KW, (1 KW *)
  • RAM Size: 64 bytes
  • Programming Mode: OTP
  • Supported by Easy PDK Programmer: yes
  • Product Page: http://www.padauk.com.tw/en/product/show.aspx?num=105&kind=41
  • Datasheet: https://www.padauk.com.cn/upload/doc/PMS15A,PMS150C datasheet_EN_V110_20230216.pdf

The PMS15A may just be a marketing name for the PMS150C. So far, all PMS15A tested appear to be identical to the PMS150C and have the same ROM size. The ROM size restriction is only enforced when using the official IDE. Of course we can’t guarantee that Padauk doesn’t change this in the future. – Source 1, Source 2
